# Things to Build - Portfolio Projects

This section details potential future projects and extensions, providing specifications for their creation.

## 1. Tech Blog Catchup - Android App

**Description:** This project aims to create an Android application that transforms technical blog posts into audio podcasts using LLM-generated scripts and TTS.

**Specification:** For detailed requirements, UI/UX, technical architecture, and API integrations, please refer to the full spec file: `memory/geordi/tech-blog-catchup/android-spec.md`

**Key Features:**
- URL input for blog posts
- LLM-powered script generation
- TTS audio synthesis
- Podcast playback with background support
- Library management

**Technology Stack:** Flutter, Riverpod, Material 3 Design, `just_audio`


## 2. OpenClaw Chrome Extension

**Description:** A Chrome extension designed to capture web content (pages, posts) and send it to OpenClaw for analysis and classification.

**Specification:** For detailed requirements, technical architecture, and classification categories, please refer to the full spec file: `memory/geordi/chrome-extension-openclaw/SPEC.md`

**Key Features:**
- Toolbar button, context menu, and keyboard shortcut for capture
- Classification into categories: Project Idea, Read Later, Blog Post, Explore AI Frontiers, Todo
- OpenClaw API integration for analysis and storage

**Status:** Specifications are complete. Development can commence.
